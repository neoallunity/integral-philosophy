# Журнал "Интегральная философия"

Система публикации научного журнала "Интегральная философия", построенная на LaTeX с использованием модульной архитектуры и автоматизированной системы сборки.

## Описание

Это полнофункциональная система издания академического журнала с:
- Модульной системой конфигурации
- Автоматической генерацией метаданных
- Комплексной системой сборки через Make
- Поддержкой библиографии, глоссариев и указателей
- Непрерывной компиляцией для удобства редактирования

## Быстрый старт

```bash
# Сборка PDF
make

# Непрерывная компиляция при изменениях
make watch

# Проверка и открытие результата
make check && make view
```
